# JS 雙人井字遊戲

井字遊戲又稱「井字棋」、「圈叉棋」、「過三關」等，英文遊戲名稱為「TIC TAC TOE」，相傳為古羅馬時代所發明，為一古老的遊戲。直到十九世紀後期，英國才開始出現圈叉棋的名稱，並演變成紙筆進行的井字遊戲。

## 使用方式
- 把整個專案複製一份到你的電腦裡，這裡指的「內容」不是只有檔案，而是指所有整個專案的歷史紀錄、分支、標籤等內容都會複製一份下來。
```sh
$ git clone
```
